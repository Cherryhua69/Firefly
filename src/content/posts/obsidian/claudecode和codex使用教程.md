# 一、最根本的两把钥匙

提升工作效率

**技能1:Skill Creator**

全局安装,自定义创建属于自己好用的工作流技能

**技能2:Find Skills**

技能自动搜索查找现成的技能去使用

# 二、具体应用场景的Skills

## 场景1:软件开发

需要的技能:(Superpowers、GStac、前端大神(mattpocock/skills))

1. **Superpowers**

    测试驱动开发(TDD)为该技能的核心驱动

    标准化工作流程:需求头脑风暴→任务精细拆解→子Agent执行

    实际用法:

    1. **brainstorming** - 头脑构思阶段编写代码前启动,通过提问梳理初步思路、探索多种实现方案,分模块输出设计方案以供审核确认,最终归档保存设计文档。
    2. **using-git-worktrees** - 多工作区分支开发阶段设计方案获批后启动,在新分支创建独立隔离工作空间,完成项目环境初始化,核验基准测试环境纯净可用。
    3. **writing-plans** - 任务规划拆解阶段依托已定版设计方案执行,将整体开发工作拆解为**2-5 分钟即可完成**的细碎任务;每项任务明确精准文件路径、完整代码内容与校验步骤。
    4. **subagent-driven-development** or **executing-plans** - 子代理驱动开发 / 方案落地执行阶段依据开发规划启动,为单个任务分配独立子执行单元,实行**两级审核**(先核对需求合规性,再核查代码质量),也可批量执行并设置人工复核节点。
    5. **test-driven-development** - 测试驱动开发(TDD)开发编码全程执行,严格遵循**红 - 绿 - 重构**流程:先编写失败测试用例→确认用例报错→编写最简实现代码→验证用例通过→提交代码;**删除所有先写代码后补测试的冗余代码**。
    6. **requesting-code-review** - 代码评审提审阶段任务间隙触发,对照开发规划逐项审核代码,按问题严重程度分类上报;高危问题直接暂停后续开发进度。
    7. **finishing-a-development-branch** - 开发分支收尾阶段全部开发任务完成后启动,全量核验测试用例,提供分支合并 / 提 PR / 保留分支 / 废弃分支多种处理方案,最后清理隔离工作区环境。
2. **GStac**

    全能专家团队阵容(23+专家角色):首席执行官、视觉设计师、后端工程师等等

    **实战全流程:**

    运行命令/office-hours,ai就会用记者模式(拷问模式)询问你关键信息

    →/plan-ceo-review(CEO高度审视计划),确保战略最优解,排除低效逻辑

    →/review, 资深工程师复核

    →(额外验证)/qa 浏览器验证

    →/ship,自动化发布;同步代码》运行测试》推送代码》开启PR

3. 前端大神(mattpocock/skills)

    解决人类和AI Agent沟通对不齐的问题:

    困境一:理解偏差(AI没听懂,废话连篇)

    困境二:执行失效(代码跑不通,逻辑不可用)

    困境三:架构隐患(缺乏规划,维护火葬场)

    **实战流程:**

    “帮我加个登录功能”→/grill-me 拷问模式开启 (得出需求精准化:SSO多租户登录)→/triage   Issue 分诊(核心阻塞任务,分清轻重缓急,细枝末节小bug)→/improve (架构急救包,站在全局视角审视代码,重构未来可能出现的问题)

    ## **场景二:前端页面设计**

    缺点:样式单一死板

    **推荐技能:**

    **1、frontend-design:现代布局,极简风格,动感交互,内置50+顶级设计规范**

    **2、ui-ux-pro-max-skill:直接配了一个设计总监,不靠直觉,靠专业设计底层逻辑,内置行业160多个风格设计细节**

    frontend-design(负责“出彩”) + ui-ux-pro-max(负责“专业”)

    ## 场景三:内容创作

    推荐技能

    **baoyu-skills**

    高质量产出、专业设计师水平封面

    文字》信息图(可视化信息图)》自动碎活(转格式,做排版)》一键发布

    生产和发布内容一体化

    里面好用的技能:

    内容技能(content skills):内容的生成和发布,极其丰富的模板

    xhs-images skill:8种排版模式,markdown格式→html页面格式

    翻译skill:分析→翻译→校正→润色

    发布微信skill、发布微博skill:一键跨平台分发
